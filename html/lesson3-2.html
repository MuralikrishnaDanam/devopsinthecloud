3.2 - Define security groups

Learning Objectives

By the end of this lesson you will be able to -

  • Create a security group that can be assigned to EC2 instances.

Create a Security Group

Create a Security Group with SSH access to port 22.

  • Go to the EC2 tab on the management console: https://console.aws.amazon.com/ec2/.
  • Select the Security Groups link on the left panel.
  • Click on the Create Security Group button and enter ssh for the name and description.
  • Select the Inbound tab and enter 22 in the Port range field.
  • Click the Add Rule button and then click the Apply Rule Changes button.
  • (Optional) Enter CIDR notation in the Source field.
